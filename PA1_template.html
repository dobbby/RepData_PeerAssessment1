Loading and preprocessing the data

Datasets for this assignment contain personal activity information during 2 months in the 5 minutes intervals. The variables included in dataset are:

  • steps: Number of steps taking in a 5-minute interval (missing values are coded as NA)
  • date: The date on which the measurement was taken in YYYY-MM-DD format
  • interval: Identifier for the 5-minute interval in which measurement was taken

What is the average daily activity pattern?

Processing data

actPattern <- aggregate(steps ~ as.factor(data$interval), data, FUN = mean)
names(actPattern) <- c("int", "steps")
actPattern$int <- as.numeric(as.character(actPattern$int))
